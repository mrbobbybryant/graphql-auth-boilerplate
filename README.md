# GraphQL Auth Boilerplate
Boilerplate contains mutations to **signup**, **login**, **logout**, and **forgot password**. Also contains a `users` query and a `me ` query for dev purposes.

## Setup
0. `npm install`
0. `docker-compose up`
0. `npm run dev`

## Envirnoment Variables
* DB_HOST
* DB_NAME
* DB_USER
* DB_PASSWORD
* JWT_SECRET
* SALT
* SENDGRID_API_KEY
* CLIENT_URL
